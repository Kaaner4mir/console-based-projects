# Konsol Tabanlı Hesap Makinesi

Bu proje, C# ile geliştirilmiş konsol tabanlı bir hesap makinesi uygulamasıdır. Kullanıcıdan giriş alarak temel aritmetik işlemlerden ileri matematiksel işlemlere kadar çeşitli hesaplamaları gerçekleştirebilir. Ayrıca hafıza işlemleri ile kullanıcı hesaplarını yönetebilir.

---

## Özellikler

* **Temel Aritmetik İşlemler:**

  * Toplama
  * Çıkarma
  * Çarpma
  * Bölme (0 ile bölme kontrolü)
* **İleri Matematik İşlemleri:**

  * Üs alma
  * Kök alma (çift dereceli köklerde negatif kontrol)
  * Faktöriyel (sadece pozitif tam sayılar)
  * Logaritma (türev ve taban kontrolü)
* **Trigonometrik İşlemler:**

  * Sinüs, Kosinüs
  * Tanjant, Kotanjant
  * Sekant, Kosekant
  * Tanımsız durumlar için hata kontrolü
* **Hafıza İşlemleri:**

  * Hafızaya ekleme
  * Hafızadan çıkarma
  * Hafızayı gösterme
  * Hafızayı sıfırlama
* **Kullanıcı Dostu Arayüz:**

  * Renkli çıktı (sonuçlar yeşil, hatalar kırmızı)
  * Geçersiz girişler için uyarı
  * İşlem tamamlandığında bilgi mesajı
* **Kolay kullanım ve anlaşılır menü sistemi**

---

## Gereksinimler

* **IDE:** Visual Studio 2022 veya üstü
* **.NET Sürümü:** .NET 6 veya üstü önerilir
* **Platform:** Windows (konsol uygulaması)

---

## Kurulum ve Çalıştırma

1. Projeyi indirin veya klonlayın.

   ```bash
   git clone https://github.com/Kaaner4mir/ConsoleBasedProjects.git
   ```
2. Visual Studio 2022 ile çözümü açın.
3. `Program.cs` dosyasını çalıştırarak uygulamayı başlatın.
4. Menüden yapmak istediğiniz işlemi seçin ve ekrandaki talimatları takip edin.

---

## Kullanım

1. Uygulamayı başlattığınızda menü ekrana gelir.
2. Yapmak istediğiniz işlemin numarasını girin.
3. Ekrandaki talimatlara göre sayıları girin.
4. İşlem sonucunu ekranda görebilirsiniz.
5. Hafıza işlemleri ile sayıları saklayabilir veya hafızayı sıfırlayabilirsiniz.
6. Çıkış yapmak için `14` numaralı seçeneği kullanabilirsiniz.
